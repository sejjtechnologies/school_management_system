How I Developed the System

The School Management System was developed using Flask (Python) for the backend, with Bootstrap for responsive and user-friendly interfaces. We integrated SQLAlchemy for database management and ensured modular design with separate routes for administrators, secretaries, and users.

The development process followed an iterative workflow: setting up environment variables securely, scaffolding atomic routes, validating database connections, and designing emotionally supportive UI/UX for clarity and empowerment.

Core Functions of the System

  • Student registration and management
  • Class and stream allocation
  • Auto-generation of admission and receipt numbers
  • Search and filter pupils by class, stream, or name
  • Real-time statistics and student counts per class
  • Secure login and role-based dashboards
  • Modular workflows for administrators, secretaries, and teachers
